주방 - 고정업무

매일해야할 고정업무

  1. 아침준비
  2. 아침,저녁 점검표 체크
  3. 제이마트 발주품목, 가격 확인
  4. 재료손질
  5. 제육 소분
  6. 제육용 야채(양파, 당근) 손질
  7. 짜계치 용 오이채, 삶은 달걀 준비
  8. 아침에 할 거 다하고 저녁 메뉴 확인 - 오래 걸리는 요리들은 먼저 해놓기. 예)닭 삶아두기, 양념해놓기, 재료 손질 등
  9. 반찬준비
  10. 쌀 씻기
    11. - 밥:1 => 물양:1.3 비율. 쌀 씻은 물 적당히 버리고 남은 물에 + 하는 것
    - 1번 => 1.3 바가지
    - 2번 => 2.6 바가지
    - 3번 => 3.9 바가지
  11. 설거지
  12. 칼 보관함 문 잠그기
  13. 마감하고 삼겹살 통에 담아두고 김치냉장고에 보관

매주 혹은 한 번씩 해야할 것

  1. 된장찌개 육수
  2. 세재 양 확인 후 발주
  3. 김치 썰기 + 묵은지 만들어 놓기 2주 숙성 텀
  4. 양짱
  5. 기름 갈아주기
  6. 행주 빨기
  7. 냉장고, 식칼 보관함, 그릇 두는 곳 닦기
  8. 마늘, 다진마늘 만들어 놓기
  9. 양배추 썰어놓기
  10. LPG가스 다 떨어지면 바로 동면가스 전화해서 시키고 교체하기. 키는 관리관실. 항상 여분 1개 두는 것

검열 올 때 해야할 것

소독

  1. 체크리스트에 있는 것들 다 열탕소독하기
  2. 아닌 것들도 이왕하는 김에 하기
  3. 플라스틱은 손상되니까 빨리 꺼내기
  4. 정비일 전날 냉장고 비우고 한 번씩 싹 닦기. 담날 최대한 깔끔하게 배치
  5. 필요없는 야채들, 양념한 야채들 그냥 버리기

체크리스트 (소독기 안에 다 있기는 하다)

  1. 칼*2 (고기 / 야채용)
  2. 도마*2 (고기 / 야채 손질용)
  3. 식기류 (숟가락, 젓가락, 포크, 나이프)
  4. 행주*1
  5. 조리병들 손*2
  6. 수질체크*2 (정수기 / 주방 수돗물)
  7. 밥주걱*1
  8. 집게*2
  9. 국자*2 (하얀색 플라스틱 2개 그거를 국자라고 하면 됨)
  10. 그릇*3
  11. 가위*1
