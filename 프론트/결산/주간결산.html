주간 결산서

  1. 주간운영결과 (표지) 1장
      - x주차 적을 때, 마지막주의 경우 4주차 or 5주차로 할 지 헷갈릴 것이다. 그 주의 수요일이 이번달인지 다음달인지 확인
      - E.G) 만약 21년 3월 31일이 수요일이면 => 4월 2일까지 포함이어도 3월 5주차 / 21년 2월 27~28일 + 3월 1~5일 => 그 주 수요일은 3월달이니까 3월 1주차

  2. 매입반품현황
      - Easy Pos 로그인 후 ➜ 상단 5번째 칸 매입발주 ➜ 리스트 4번째 칸 매입관리 ➜ 리스트 4번째의 거래선 상품별 매입현황 ➜ 일주일 설정(저번 주 토요일~당일 금) ➜ 조회 ➜ 엑셀 파일 받기
      - 다운로드 폴더 열고, 받은 파일을 드래그 해서 우리 결산서 쓸 떄 쓰는 '2010년' 엑셀로 집어넣기. 2011로 바로 열면 편집 불가능. ➜ 맨 왼쪽 위 모서리 눌러서 전체 선택 후, 열과 열 사이 눌러서 간격 줄이기 ➜ 맨끝 L열 지우기 + 윗줄 F~I열까지 드래그 후 지우기. 다른 곳 클릭 되어 있으면 오류남 행만 클릭 delte
      - 액셀 위쪽 구석 파일 - 최근에 사용한 항목 누르면 탑 5개가 우리가 주로 쓰는 것 나온다. 저번주에 쓴 '상품별 매입반품 양식' 연다
      - 날짜 바꾸고 방금 수정한 이번주 파일을 복붙한다. 식당에서 부터 끝까지 여분 포함해서 드래그하고 아마 복붙이 실패할 수도. 그러면 한 번 더 복사. 복사 두 번하면 가능
      - 여분포함해서 복사 후, 마우스 우클릭 ➜ 주변서식에 맞게 붙여넣기 하면, 저번주보다 매입현황이 적든 많든 깔끔하게 복붙 된다.
      - 이 액셀 파일과 1주일간 적은 원재료비 한글 파일 한 장 씩 인쇄

  3. 상품별 일 매출(여기서 부터는 마감 누르기 전에)
      - Easy Pos 로그인 후 ➜ 상단 3번째 칸 매출분석 ➜ 리스트 2번째 칸 상품분석 ➜ 리스트 5번째의 상품별 일 매출분석 ➜ 일주일 설정(저번 주 토요일~당일 금) + 하단의 중or소 분류소계설정(파란색 색깔 나타남 = 우리가 따로 안해도 됨) ➜ 조회 ➜ 액셀 파일 받기
      - 다운로드 폴더 열고, 받은 파일을 드래그 해서 우리 결산서 쓸 떄 쓰는 '2010년' 엑셀로 집어넣기. 2011로 바로 열면 편집 불가능. ➜ 맨 왼쪽 위 모서리 눌러서 전체 선택 후, 열과 열 사이 눌러서 간격 줄이기 ➜ 앞글자 대/상/바/총/총 만 두고, 중~소 분류 드래그 지우기, 나머지 총이후 쭉 지우기. 2번 나눠서 해야 잘 지워짐
      - 날짜 바꾸고 방금 수정한 이번주 파일을 복붙한다. 식당에서 부터 끝까지 여분 포함해서 드래그하고 아마 복붙이 실패할 수도. 그러면 한 번 더 복사. 복사 두 번하면 가능
      - 여분포함해서 복사 후, 마우스 우클릭 ➜ 주변서식에 맞게 붙여넣기 하면, 저번주보다 매입현황이 적든 많든 깔끔하게 복붙 된다.
      - 인쇄 + 추가로 포스기에서 [특정기간 상품내역 조회] ➜ 같은 일주일 설정해서 뽑기 ➜ 금액 똑같아야함 ➜ 주간운영결과 표지 앞에 스테이플러

  4. 승인 관리
      - Easy Pos 로그인 후 ➜ 상단 4번째 칸 승인관리 ➜ 리스트 1번째 칸 신용카드 ➜ 리스트 1번째의 일자별 승인 현황 클릭 ➜ 일주일 설정(저번 주 토요일~당일 금) ➜ 조회 ➜ 엑셀 파일 받기
      - 다운로드 폴더 열고, 받은 파일을 드래그 해서 우리 결산서 쓸 떄 쓰는 '2010년' 엑셀로 집어넣기. 2011로 바로 열면 편집 불가능. ➜ 맨 왼쪽 위 모서리 눌러서 전체 선택 후, 열과 열 사이 눌러서 간격 줄이기 ➜ 전체적으로 깔금하게 찌뿌된 것 조금씩 늘려준다 (특히 승인일자 ### 안 나오게, 승인 금액)
      - 인쇄단축키(ctrl+p) 해서 인쇄 설정쪽을 보면 맨 밑에 '현재 설정된 용지' 있는데 이 것을 '한 페이지에 시트 맞추기'로 변경 그 후 인쇄. 두 장 나와도 상관없으니 큼직하게 잘 보이게 하면 됨.

  5. 마무리
      - 주간운영 결산서 최종본은 특정기간 상품내역 조회 + 주간운영결과표지 + 상품별 일 매출 + 매인반품현황 + 원재료비 + 승인 관리 순으로 포개어 놓고 스테이플러 찍기
      - 여기에다가 일단 관리관님한테 일일결산 항목들 + 근무자명령서 다 싸인 받고, 최종적으로 주간운영 결산서 + 영수증철 + 일일 결산서 순으로 포개어 놓고 서랍안에 있는 클립으로 찝고 제출
